# Disk Space Inspector v0.5.0-preview.1

First public preview focused on making the app easy to try.

## Downloads

- `DiskSpaceInspectorSetup-0.5.0-preview.1.exe` - per-user Windows installer.
- `DiskSpaceInspector-0.5.0-preview.1-win-x64.zip` - portable self-contained build.
- `SHA256SUMS.txt` - checksums for release artifacts.

## Highlights

- First-run welcome workspace with `Scan this PC`, `Open demo workspace`, and cleanup safety guidance.
- Demo workspace can be loaded from inside the app without scanning local drives.
- 5-minute findings panel summarizes safe reclaimable space, ownership signals, growth, scan gaps, and AI readiness.
- Privacy & Safety Center documents no telemetry, local database storage, Codex credential policy, and blocked cleanup paths.
- Local diagnostics export writes Markdown and JSON with user-profile paths redacted by default.

## Notes

This preview is unsigned. Windows SmartScreen may show a reputation warning until code signing is added.

Cleanup remains advisory and staged. This release does not directly delete files.
