# Disk Space Inspector v1.0.0

Stable Windows release focused on local storage analysis, safer cleanup review, and a polished visual workbench.

## Downloads

- `DiskSpaceInspectorSetup-1.0.0.exe` - per-user Windows installer.
- `DiskSpaceInspector-1.0.0-win-x64.zip` - portable self-contained build.
- `SHA256SUMS.txt` - checksums for release artifacts.

## Highlights

- Stable first-run flow with `Scan this PC`, `Open demo workspace`, and cleanup safety guidance.
- Local-only Cleanup Advisor driven by scan evidence, deterministic rules, and blocked-path guardrails.
- Visual Lab card layout and chart renderer updated to prevent label and description overlap.
- Typography pass using a calmer Windows operations UI scale.
- Privacy & Safety Center documents no telemetry, local database storage, local diagnostics export, and no external advisor service.

## Notes

This build is unsigned. Windows SmartScreen may show a reputation warning until code signing is added.

Cleanup remains advisory and staged. This release does not directly delete files.
