# Disk Space Inspector v1.1.0

Stable Windows release focused on consumer clarity, ownership correction, settings, cleanup review workflow, and safer public messaging.

## Downloads

- `DiskSpaceInspectorSetup-1.1.0.exe` - per-user Windows installer.
- `DiskSpaceInspector-1.1.0-win-x64.zip` - portable self-contained build.
- `SHA256SUMS.txt` - checksums for release artifacts.

## Highlights

- Corrects ownership and license language to Andrew Rainsberger.
- Adds explicit use-at-your-own-risk language in the license, README, docs, and Settings.
- Renames confusing app areas to clearer consumer labels: Files, Map, Charts, What Changed, Explanations, and Guide.
- Adds a real Settings area for startup behavior, scan defaults, chart density, report privacy, scan history, diagnostics, and local data controls.
- Adds a cleanup review queue with exact paths, evidence, size, safety lane, removal, clearing, and local checklist export.
- Adds useful file-hunt shortcuts for biggest folders, safest cleanup, downloads, developer bloat, system cleanup routes, app-owned storage, and recent growth.
- Improves What Changed so a first scan is treated as the baseline instead of implying every scanned file is new.
- Updates chart labels and tutorials to be understandable without knowing analytics jargon.

## Notes

This build is unsigned. Windows SmartScreen may show a reputation warning until code signing is added.

Cleanup remains advisory and staged. This release does not directly delete files. Review paths carefully and back up important files before taking cleanup action outside the app.
