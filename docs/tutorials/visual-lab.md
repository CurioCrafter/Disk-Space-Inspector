# Visual Lab

Visual Lab turns a loaded scan into 24+ chart types without static mock values.

Start with `Best insights` for the highest-signal graphs:

- storage intelligence summary
- drive capacity rings
- top folder Pareto
- cumulative size curve
- extension rank
- cleanup effort/value matrix

Open `Advanced algorithms` for ownership flows, package-manager footprint, cloud/local status, entropy, stale-data scoring, and system/dev root analysis. Chart tooltips show evidence and values; node-backed chart items sync to the inspector.
