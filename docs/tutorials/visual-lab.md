# Charts

Charts turns a loaded scan into 24+ storage views without static mock values.

Start with `Best insights` for the highest-signal graphs:

- storage summary
- drive fullness
- biggest folders
- largest items curve
- file types by size
- cleanup value

Open `More charts` for app ownership, package caches, cloud/local storage, folder mix, stale-file scoring, developer caches, and protected system areas. Chart tooltips show evidence and values; node-backed chart items sync to the inspector.

Use Charts to decide what deserves attention, then use Cleanup or Files to review exact paths.
