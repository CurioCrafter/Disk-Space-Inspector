# First Scan

1. Launch Disk Space Inspector.
2. Pick a drive in the top command bar.
3. Use `Scan selected` for one drive or `Scan all` for all ready fixed/removable drives.
4. Watch accounted bytes, throughput, ETA, queue depth, and scan gaps.
5. Open `Files` after results begin loading to inspect the visible subtree.
6. Run a second scan later to populate `What Changed`.

Disk Space Inspector runs unelevated by default. Permission gaps are recorded as scan issues instead of hidden.
